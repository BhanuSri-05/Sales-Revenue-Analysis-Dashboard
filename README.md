# Sales & Revenue Analysis Dashboard

## Project Overview

This Power BI dashboard analyzes sales performance, customer behavior, profitability, and regional trends using the Superstore dataset.

## Tools Used

- Power BI
- Power Query
- DAX
- Excel/CSV

## KPIs

- Total Revenue
- Total Profit
- Total Orders
- Total Customers
- Average Order Value
- Profit Margin %

## Dashboard Features

- Revenue Trend Analysis
- Product Performance
- Regional Analysis
- Customer Analysis
- Interactive Slicers

## Dashboard Preview

![Dashboard](dashboard.png)

## Key Insights

- Technology generates the highest revenue.
- Central region contributes the most sales.
- Top customers generate significant revenue.
- Profit margin remains positive across categories.
